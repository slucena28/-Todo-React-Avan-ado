# 📝 Todo React Avançado

Aplicação de lista de tarefas desenvolvida com React, utilizando conceitos avançados como Context API, Hooks customizados e memoização.

## 🚀 Tecnologias

- React
- JavaScript
- Context API
- Hooks (useState, useEffect, useMemo)
- LocalStorage

## 📦 Funcionalidades

- Adicionar tarefas
- Marcar como concluída
- Remover tarefas
- Filtrar tarefas
- Persistência no localStorage

## ▶️ Como rodar o projeto

```bash
npm install
npm run dev
